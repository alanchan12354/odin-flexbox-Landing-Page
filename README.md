# Odin Flexbox Landing Page

A Foundations course project from [The Odin Project](https://www.theodinproject.com/) that recreates a marketing landing page using semantic HTML and Flexbox. The page features a hero section, services grid, showcase gallery, testimonial, and call-to-action card.

## Getting started

```bash
git clone <repo>
cd odin-flexbox-Landing-Page
open index.html # or use your preferred live server
```

## Tech stack

- HTML5 with semantic structure
- CSS3 with Flexbox layout utilities and responsive enhancements
- Google Fonts (Roboto)
